# Chicago Taxi Analytics Pipeline

## Tech Stack
- Google BigQuery
- PySpark 4.0
- Python + Pandas
- Matplotlib
- Google Colab

## Dataset
500k+ Chicago taxi trips from GCP public dataset

## Key Questions
1. Which company generates most revenue?
2. Does payment type affect tipping?
3. Which pickup area has highest average fare?

## Note
Results based on 500k row sample from 180M+ total trips.
